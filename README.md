# Wings Of Discovery

Simple local website made by kids to explore Drone Mapping and AI Image Detecting.

Files:

- `index.html` — Home page and navigation
- `drone-mapping.html` — Drone Mapping explanation and ideas
- `ai-image-detecting.html` — AI Image Detecting explanation and ideas
- `about.html` — About the project
- `css/style.css` — Simple styles

Open locally:

On Windows, double-click `index.html` in the folder or right-click -> Open with -> your browser. No deployment or server is required.
